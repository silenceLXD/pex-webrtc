# pex-webrtc

## 简介
  基于网页的视频会议，实现了会议远端画面、本地画面、共享、摄像头、语音、扬声器、参会者、会议操作等
  
## 目录
  ```
  static/             --> 引用文件
  v20/                --> v20版本相关代码
    app/                 --> 项目的源文件
      css/                  --> css文件
      js/                   --> js文件
      templates/            --> 视图模板
      widgets/              --> 第三方引用文件
      index.html            -->项目布局模板
  web/                --> v19 build后
  ```
  
## 说明
  此项目仅供参考，不可用于任何商业
